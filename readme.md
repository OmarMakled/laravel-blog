## Install

```shell
git clone git@github.com:OmarMakled/laravel-blog.git
```

```shell
cp .env.example .env
```

edit the `.env` file and set the `database` and other config for the system after you copy the `.env`.example file.

```shell
composer install
```

```shell
php artisan migrate --seed
```
